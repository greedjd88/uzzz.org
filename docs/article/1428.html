npm安装

npm i element-ui -S

引入 Element
你可以引入整个 Element，或是根据需要仅引入部分组件。我们先介绍如何引入完整的 Element。
在 main.js 中写入以下内容：

import Vue from 'vue';
import ElementUI from 'element-ui'; //完整引入element-ui这个组件
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});

上面就相当于可以使用ElementUI这个组件了
